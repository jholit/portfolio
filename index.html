Bookmark Side Panel — UI Modernization

Product Design

A product design exploration focused on improving how users view, scan, and organize bookmarks through clearer structure and reduced visual noise.

  • Clear category and subfolder hierarchy
  • Faster visual scanning and recognition
  • Reduced eye strain through softer visual emphasis
  • Improved structural clarity without adding complexity

Bookmark panels are used constantly but often feel cluttered or fatiguing over time. This project refines the panel’s hierarchy and visual balance, making saved items easier to recognize and manage while preserving familiarity and user control.

ChatGPT — Structural Interface Redesign

Product Design

A conversation-first interface redesign focused on reducing cognitive friction by separating navigation from communication and preserving the conversation as the primary workspace.

  • Distraction-minimal conversation space
  • Clear separation of navigation and communication
  • Predictable, consistent interface structure
  • Foundational layout for future feature scalability

Many conversational tools expose system structure in ways that interrupt flow and increase mental overhead. This redesign preserves uninterrupted conversation by aligning the interface with natural user thinking.

TennaJam — Simplifying Complex Technical Workflows

Product Design

A systems-focused design project demonstrating how a dense, multi-step technical process can be restructured into a guided, predictable workflow.

  • High-level workflow simplification
  • Progressive disclosure of complex configuration
  • Clear separation of overview and control surfaces
  • Structured state modeling for confidence and clarity

TennaJam restructures technical complexity into manageable, step-based flows that reduce cognitive load while preserving system power.
