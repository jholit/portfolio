Memory Machine – Cognitive Architecture for AI

A deep dive into how long-term AI memory breaks down over time—specifically the loss of context and fragmentation that occurs across sessions. This study explores a bio-inspired cognitive and technological memory system designed for modern language models.

  • Ecosystem-like memory nodes
  • Retention logic & decay behaviors
  • Context weighting and arbitration
  • Long-term continuity constraints

I approached this by treating memory less like a static database and more like a living ecosystem, where continuity, decay, and relevance are intentional behaviors. This led to a clearer model for long-term system behavior and reduced fragmentation over time.

TennaJam – Game Server & Management Application

A project close to my heart focused on making game server setup and management more approachable, guided, and forgiving—even for users who don’t speak “DevOps.” This is how dedicated gaming servers should feel in the modern era.

  • Simplified server creation flows
  • Region & setting clarity
  • Soft-surfaced advanced options
  • Clean management dashboard

Most server tools assume users already understand DevOps concepts, which creates unnecessary friction and intimidation. I designed TennaJam with a surface-level mindset for each step, while making advanced options available only when needed. This turned server setup into a guided journey instead of a daunting or inaccessible task.

Quillty – Writing & Notetaking Application

A writing space designed to reduce cognitive friction through calm structure, thoughtful typography, and clear drafting flows. Built for both everyday and professional writing, with room to grow.

  • Distraction-minimal UI
  • Soft, readable typography
  • Simple drafting workflows
  • Foundational structure for future expansion

Many writing tools optimize for features at the expense of mental clarity. I designed Quillty around calm structure and low-friction drafting flows to reduce cognitive load. This helps writing feel more continuous and less interrupted by interface decisions.

Homepage Redesign – Renegade Game Studios

A Personal Audit in Low-Fidelity Webpage Restructuring

A redesign of the Renegade Game Studios homepage! Right now, it tries to do too much at once, which makes it harder for users to quickly understand what the site is actually for. Even as someone familiar with the brand, I often find myself avoiding the website because the interactions feel confusing and the overall experience lacks focus.

This redesigned layout is about getting back to the basics—reorienting the homepage around clarity, confidence, and intent. The goal is to improve UI/UX at the root, reduce friction, and increase user retention, session depth, and engagement by making it immediately clear "why" someone is there and what they can do next. If you want a comparison point, you can check out their current website.
