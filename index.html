TennaJam – Game Server & Management Application

Product Design

A product design case study focused on making game server setup and management more approachable and guided for non-technical and semi-technical users.

  • Simplified server creation flows
  • Region & setting clarity
  • Soft-surfaced advanced options
  • Clean management dashboard

Most server tools assume users already understand DevOps concepts, creating unnecessary friction. TennaJam reframes server setup as a guided flow, surfacing advanced options only when needed and reducing intimidation during critical setup steps.

Quillty – Writing & Notetaking Application

UX Research (Prototype/Concept)

A writing and note-taking concept designed to reduce cognitive friction through calm structure, readable typography, and clear drafting workflows.

  • Distraction-minimal UI
  • Soft, readable typography
  • Simple drafting workflows
  • Foundational structure for future expansion

Many writing tools prioritize features over mental clarity. Quillty focuses on low-friction drafting flows that help writing feel continuous and uninterrupted by interface decisions.

Memory Machine – Cognitive Architecture for AI

Case Study

A case study exploring how long-term AI memory degrades over time, particularly through context loss and fragmentation across sessions.

  • Ecosystem-like memory nodes
  • Retention logic & decay behaviors
  • Context weighting and arbitration
  • Long-term continuity constraints

This study treats memory as a dynamic system rather than a static database, emphasizing continuity, decay, and relevance as intentional behaviors within long-term system design.

Homepage Redesign – Renegade Game Studios

An Audit in Low-Fidelity Webpage Restructuring

A redesign of the Renegade Game Studios homepage, addressing clarity issues caused by competing priorities and unclear content hierarchy. It's current iteration tries to do too much at once, which makes it harder for users to quickly understand what the site is actually for. Even with brand familiarity, the current experience makes it difficult to quickly understand intent and next steps.

This redesigned layout is about getting back to the basics — reorienting the homepage around clarity, confidence, and intent. The goal is to improve UI/UX at the root, reduce friction, and increase user retention, session depth, and engagement by making it immediately clear "why" someone is there and what they can do next.
